# AI-Based Fake Job Detection

A machine learning project that identifies fraudulent job postings using Natural Language Processing (NLP) and Deep Learning techniques.

## Features
- Detects fake and genuine job advertisements
- Text preprocessing and feature extraction
- Deep learning-based classification
- Performance evaluation using accuracy and loss metrics

## Technologies Used
- Python
- TensorFlow / Keras
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Dataset
Fake Job Postings Dataset from Kaggle containing real and fraudulent job advertisements.

## Results
The model analyzes job posting content and predicts whether a job listing is **Real** or **Fake**.

## Future Enhancements
- Streamlit/Flask deployment
- BERT-based NLP models
- Real-time job fraud detection
